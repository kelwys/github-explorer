## 🚀 Getting Started

First, run the development server:

```bash
# Clone Repository
$ git clone https://github.com/kelwys/github-explorer.git

# Go to server folder
$ cd github-explorer

# Install Dependencies
$ yarn

# Run Aplication
$ yarn start

# Access localhost
http://localhost:3000
```
